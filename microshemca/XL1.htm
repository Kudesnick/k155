К155ХЛ1, КМ155ХЛ1

Микросхемы К155ХЛ1, КМ155ХЛ1 представляют собой универсальные элементы для ЦВМ, реализующие функции хранения и обработки информации.

Содержат два базовых элемента, каждый из которых включает D-триггер с тремя информационными входами и три шины магистрали.

Микросхемы К155ХЛ1, КМ155ХЛ1 могут быть использованы для построения програмно-адресуемых регистров, регистров хранения данных и промежуточных результатов, мультиплексоров-демультиплексоров данных и других устройств.

Микросхемы К155ХЛ1, КМ155ХЛ1 не имеют зарубежных аналогов. Они представляет собой D-триггер с тремя информационными входами-выходами. Для записи информации в D-триггер имеются три входа для управляющих сигналов записи, каждый вход записи для «своего» входа-выхода. Аналогично для чтения информации из D-триггера имеются три входа для управляющих сигналов чтения. Микросхемы К155ХЛ1, КМ155ХЛ1 содержат два D-триггера с общими управляющими входами записи-чтения и раздельными информационными входами выходами. Два триггера позволяют реализовать два разряда регистра.
