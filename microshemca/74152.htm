74152

Описание

Микросхемы 74152, 74LS152 содержат селектор данных, который из восьми входных сигналов выделяет один с помощью 3-разрядного двоичного кода.

Работа схемы

Необходимый входной сигнал (DO — D7) выбирается подачей на адресные входы микросхем 74152, 74LS152 (АО — А2) соответствующего двоичного кода. Выделенный сигнал появляется на выходе Q в инвертированном виде.

Микросхемы 74152, 74LS152 можно использовать в качестве функционального преобразователя, а также для последовательной передачи данных.

Применение

Мультиплексор, дешифратор адреса, последовательная передача данных, функциональный преобразователь.

Производится следующая номенклатура микросхем: 74152,74LS152.
