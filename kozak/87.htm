ИР15 = (173) - 4-разрядный параллельный регистр с тремя состояниями

G1,G2 управление записью
 M,N  управление выходами
  Cк  тактирование
  Cl  сброс

4-разрядный регистр с выходом с 3-мя состояниями, с запретом записи и асинхронным сбросом. (Счетверенный D-триггер).
 Cl  Cк  G1 G2  D  Q
 H   X  X  X  X  L
 L   L  X  X  X  Q0
 L  _/~  H  X  X  Q0
 L  _/~  X  H  X  Q0
 L  _/~  L  L  L  L
 L  _/~  L  L  H  H
Если "M" или "N" в состоянии H, то выходы регистра переводятся в состояние с высоким импедансом.
